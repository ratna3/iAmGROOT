# Changelog

All notable changes to **We are GROOTS** will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2025-01-21

### Added

- Initial release of We are GROOTS AI Chatbot
- Puter.js integration for free unlimited Claude API access
- Support for multiple Claude models:
  - claude-sonnet-4-5 (default)
  - claude-opus-4-5
  - claude-haiku-4-5
  - gpt-4o
  - gpt-4o-mini
  - And more...
- Supabase backend integration
  - User authentication (anonymous)
  - Conversation persistence
  - Message history storage
- Modern, responsive UI
  - Dark/Light theme toggle
  - Mobile-first design
  - Smooth animations
- Conversation management
  - Create new conversations
  - Search/filter conversations
  - Delete conversations
- Message features
  - Real-time streaming responses
  - Regenerate last response
  - Code syntax highlighting
  - Markdown rendering
- Groot branding
  - Custom logo (Baby Groot)
  - 3D animated Groot model (Three.js)
- Netlify deployment configuration
- Comprehensive documentation

### Security

- Row Level Security (RLS) on all database tables
- User data isolation
- Secure API handling through Puter.js
